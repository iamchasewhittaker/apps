## Example tickets

### Title
Set up iOS project shell

### Description
Create Xcode project, repo structure, bundle id, and initial tab navigation scaffold.

### Acceptance criteria
- Xcode project builds successfully
- repo is created
- basic folder structure exists
- placeholder tabs render on simulator

### Dependencies
- app name chosen
- bundle id chosen

### Estimate
M

### Status
Not Started

### Title
Build create-item flow

### Description
Implement the screen and logic for creating a new item.

### Acceptance criteria
- user can open create flow
- user can enter required fields
- validation appears if required fields are empty
- item is saved locally
- item appears in the main list

### Dependencies
- data model exists
- persistence layer exists

### Estimate
M

Not Started
