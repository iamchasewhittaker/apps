# ClarityUI

Shared Swift package providing colors, typography, spacing, and reusable SwiftUI components for the Clarity iOS app family.

## Stack
- Swift 5.9+ / SwiftUI
- Swift Package Manager (local SPM dependency)
- No external dependencies

## Build
```bash
swift build
swift test
```

Or reference as a local SPM dependency from any Clarity iOS app's `.xcodeproj` via `../clarity-ui`.
