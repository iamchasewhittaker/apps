# RollerTask Tycoon — planning (filled artifacts, iOS)

**Park-specific** product and engineering docs. **Canonical blank templates** stay in the monorepo **[iOS App Starter Kit](../../../../docs/ios-app-starter-kit/README.md)** (`docs/ios-app-starter-kit/`).

## How to use

1. Read **[PLANNING_WORKFLOW.md](PLANNING_WORKFLOW.md)** before medium-or-larger work.
2. Keep these files aligned with what you ship; update **PRD** / **App Flow** / **Technical Design** when scope changes.
