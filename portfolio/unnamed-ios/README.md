# Unnamed iOS

Native iOS daily OS for ADHD brains — 4 fixed lanes, lane lock, one-at-a-time focus. Mirrors the web app flows exactly with iOS-native haptics and swipe gestures.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults + JSON Codable
- No external dependencies
- Local Xcode build

## Build
Open `Unnamed.xcodeproj` in Xcode 15+ and run on device or simulator.
