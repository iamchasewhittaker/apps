# Clarity Command iOS

Native iOS daily accountability hub — morning mission commit, evening reflection, scoreboard streaks, and faith-driven urgency. Syncs to the same Supabase blob as the web app.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults + JSON Codable / Supabase Swift SPM
- ClarityUI shared package (local SPM)
- Local Xcode build

## Build
Open `ClarityCommand.xcodeproj` in Xcode 15+ and run on device or simulator.
