# Clarity Check-in iOS

Native iOS morning and evening wellness check-in app with mood logging, medication tracking, pulse snapshots, and daily quotes.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults + JSON Codable
- ClarityUI shared package (local SPM)
- Local Xcode build

## Build
Open `ClarityCheckin.xcodeproj` in Xcode 15+ and run on device or simulator.
