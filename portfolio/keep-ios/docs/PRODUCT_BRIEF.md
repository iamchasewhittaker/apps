# Product Brief — Keep

> Phase 1: Product Definition.

## App summary
Keep helps Chase organize his home by triaging every item (keep, donate, toss, or unsure) and assigning kept items a permanent spot.

## Target user
Chase (primary), Kassie (eventually). ADHD brain that freezes on decluttering decisions and has no organizational system.

## Main pain point
Stuff everywhere with no system. Can't find things, can't let go, nothing has a designated home. Kassie is frustrated.

## Core value
Turns the overwhelming question "what do I do with all this stuff?" into a one-item-at-a-time triage flow with an optional decision coach. Every kept item gets an address.

## V1 scope
- Create rooms and define spots within each room
- Add items via camera photo + name
- Triage items one at a time: Keep / Donate / Toss / Unsure
- Assign kept items to spots
- Progress stats and donation bag counter
