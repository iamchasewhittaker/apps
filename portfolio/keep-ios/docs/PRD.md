# Product Requirements — Keep

> Phase 2: PRD.

## Problem
Chase has stuff everywhere — garage, closets, multiple rooms — with no organizational system. Items sit on counters because there's no designated spot. He freezes or overthinks when deciding what to keep. The chaos causes friction with Kassie.

## Users
- Chase (primary) — ADHD, hoarder tendencies, needs a system not a one-time purge
- Kassie (Phase 2) — would use shared data to coordinate household organization

## User stories
- As a user, I want to create rooms and spots so I can define where things belong
- As a user, I want to photograph items quickly so I have a visual record
- As a user, I want to triage one item at a time so I don't get overwhelmed
- As a user, I want help deciding on hard items so I don't freeze
- As a user, I want to see my progress so I stay motivated
- As a user, I want to track donation bags so I can see how much I've let go

## V1 features
- Room management (create with name + emoji)
- Spot management (create within rooms — e.g., "Shelf A", "Under workbench")
- Item entry (camera photo + name, batch mode)
- Triage flow (one card at a time: Keep / Donate / Toss / Unsure)
- Spot picker (assign kept items to a specific spot)
- Decision coach (triggered after 3+ Unsure — 3 yes/no questions, soft suggestion)
- Stats dashboard (totals, breakdown by status, room progress, donation bags)
- Warm amber theme, dark mode only

## Not in V1
- Search ("where is my camping stove?")
- Barcode scanning
- Categories or tags
- Kassie access / shared data
- Web companion
- Export donation list
- iCloud sync
- Notifications or reminders
- "New item arrives" quick-add from home screen

## Success metrics
- Chase can triage one room end-to-end in a single session
- The app is used at least 3 times in the first week
- At least 1 donation bag is filled and counted
