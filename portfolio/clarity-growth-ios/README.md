# Clarity Growth iOS

Native iOS habit tracker covering seven growth areas with streak-aware session logging and history review.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults + JSON Codable
- ClarityUI shared package (local SPM)
- Local Xcode build

## Build
Open `ClarityGrowth.xcodeproj` in Xcode 15+ and run on device or simulator.
