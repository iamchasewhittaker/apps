# Clarity Time iOS

Native iOS focus timer and time logger paired with a scripture completion streak — records timed or manually entered work sessions alongside daily scripture reading.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults + JSON Codable
- ClarityUI shared package (local SPM)
- Local Xcode build

## Build
Open `ClarityTime.xcodeproj` in Xcode 15+ and run on device or simulator.
