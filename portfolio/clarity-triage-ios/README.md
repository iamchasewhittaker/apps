# Clarity Triage iOS

Native iOS daily triage app built around energy-based capacity slots — pick today's energy level, assign weighted tasks, move ideas through a pipeline, and log wins.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults + JSON Codable
- ClarityUI shared package (local SPM)
- Local Xcode build

## Build
Open `ClarityTriage.xcodeproj` in Xcode 15+ and run on device or simulator.
