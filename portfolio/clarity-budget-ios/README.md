# Clarity Budget iOS

Dual-scenario iOS budget with live YNAB integration — shows safe-to-spend figures for month, week, and day, with category import and fund-assignment via YNAB API.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults + JSON Codable / YNAB token in Keychain
- ClarityUI shared package (local SPM)
- Local Xcode build

## Build
Open `ClarityBudget.xcodeproj` in Xcode 15+ and run on device or simulator.
