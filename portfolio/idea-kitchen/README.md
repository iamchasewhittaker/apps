# Idea Kitchen

A reusable system for turning raw ideas into shipped portfolio apps. Pairs a Claude Project prompt (for ideation, naming, pressure-testing, and artifact generation) with a Claude Code handoff checklist and a per-app minisite.

**Start here:** [`docs/BUILD_GUIDE.md`](docs/BUILD_GUIDE.md).

The Claude Project system prompt lives at [`prompts/CLAUDE_PROJECT_IDEA_KITCHEN.md`](prompts/CLAUDE_PROJECT_IDEA_KITCHEN.md). Upload the files listed in [`templates/CLAUDE_PROJECT_KNOWLEDGE_MANIFEST.md`](templates/CLAUDE_PROJECT_KNOWLEDGE_MANIFEST.md) as Project Knowledge.
