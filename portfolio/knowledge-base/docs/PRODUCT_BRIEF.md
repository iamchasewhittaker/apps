# Knowledge Base — Product Brief

> Phase 1 of the [Product Build Framework](../../PRODUCT_BUILD_FRAMEWORK.md)

## App Summary

This app helps Chase store and quickly reach every link he needs — AI docs, his own projects, and general reference — in one fast page.

## Target User

Chase (personal use). A developer who saves links across browsers, notes, and chat windows and needs one central, fast page to reach everything.

## Main Pain Point

Links are scattered across browsers, notes, and AI chats — there is no single source of truth for bookmarks that matter.

## Core Value

One page, instant access, everything in one place. No hunting, no switching contexts. Open the page, search or filter, click the link.

## V1 Scope

1. Dark mode single-page UI
2. Search across title / URL / category
3. Category filter pills (dynamic from data)
4. Add / edit / delete bookmarks
5. Persistent storage (`window.storage` — Claude artifact environment)
6. Seed data for AI docs (22 entries across 8 categories)
7. "My Projects" category with Chase's active portfolio apps
