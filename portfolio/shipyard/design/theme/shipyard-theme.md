# Shipyard

A dark nautical command theme — deep navy infrastructure meets precision engineering. Evokes chart rooms, compass bearings, and the quiet authority of fleet command. Every detail placed with painstaking care, as if charted by an expert navigator.

## Color Palette

- **Deep Navy**: `#07101E` — Primary background; the ocean at 0200 hours
- **Warm White**: `#F2EEE6` — Primary text and icons; worn chart paper
- **Steel Blue**: `#4A90DE` — Accent; bearing indicators, active states
- **Gold**: `#D7AA3A` — Highlight rule; compass rose gold
- **Dim Blue**: `#346090` — Secondary text; faded nautical markings
- **Ghost Navy**: `#0D1C36` — Subtle rules and grid; background texture
- **Muted Navy**: `#1A2E52` — Card / surface backgrounds

## Typography

- **Headers / Wordmark**: BigShoulders Bold — industrial, wide, confident
- **Body / Labels**: DM Mono Regular — monospaced, technical, precise
- **Secondary**: DM Mono Regular (at reduced opacity or size)

## Design Principles

- Background grid of faint horizontal rules (every 40px) to evoke graph paper / nautical charts
- Compass ring motifs (ghost circles + cardinal tick marks) as ambient decoration
- Steel blue bearing-arc fragments for emphasis — never full circles, always partial
- Gold used sparingly: single underlines, single accent marks only
- All type at or near the left edge of a clear vertical-rule boundary
- Sonar / radar arcs (partial) in extreme corners to frame compositions
- Monospaced coordinates (lat/long, version strings) as micro-detail anchors

## Usage

Dark-mode dashboards, developer tool UIs, nautical / maritime products, portfolio showcases, command-and-control interfaces, technical documentation covers.
