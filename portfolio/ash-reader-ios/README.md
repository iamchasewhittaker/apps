# Ash Reader iOS

Native SwiftUI companion to Ash Reader — the full 138k-word capture system conversation baked in, with four tabs (Reader, Themes, Actions, Settings), iCloud sync, reminders, and streak tracking.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults for progress persistence (`ash_reader_ios_` prefix)
- Fully offline — no backend, no network
- Local Xcode build

## Build
Open `AshReader.xcodeproj` in Xcode 15+ and run on device or simulator.
