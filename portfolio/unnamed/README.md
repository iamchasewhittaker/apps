# Unnamed

Daily OS for ADHD brains. 4 fixed lanes (Regulation, Maintenance, Support Others, Future), lane lock, one-at-a-time focus.

## Stack
- Next.js 16 + TypeScript + Tailwind CSS 4
- localStorage (offline-first)
- Deployed: unnamed-gold.vercel.app

## Run locally
```bash
npm install && npm run dev
```

## 7-Day Rule
No new features until 7-day consecutive usage streak is achieved.
