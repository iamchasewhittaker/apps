# Funded iOS

Native iOS YNAB companion app answering four questions the YNAB UI answers poorly: bills coverage, shortfall, safe-to-spend, and mortgage funding timeline. Reads and writes via the YNAB API with Keychain-secured token.

## Stack
- Swift 5.0+ / SwiftUI / SwiftData / iOS 17
- Keychain for YNAB token storage
- No third-party dependencies
- Local Xcode build

## Build
Open `Funded.xcodeproj` in Xcode 15+ and run on device or simulator.
