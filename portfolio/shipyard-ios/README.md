# Shipyard iOS

Native iOS companion to Shipyard web — a read-only fleet command center pulling portfolio data from Supabase with email/password auth and offline cache fallback.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults cache + supabase-swift v2
- Local Xcode build

## Build
Open `Shipyard.xcodeproj` in Xcode 15+ and run on device or simulator.
