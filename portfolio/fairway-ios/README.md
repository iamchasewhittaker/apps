# Fairway iOS

Full lawn management OS for a 4-zone Rachio irrigation system — covers zones, sprinkler heads, problem areas, fertilizer plan, soil test data, spreader calculator, shrub beds, maintenance tasks, mow log, and live Rachio API integration.

## Stack
- Swift 5.9+ / SwiftUI / iOS 17
- UserDefaults + JSON Codable / Rachio token in Keychain
- No external dependencies
- Local Xcode build

## Build
Open `Fairway.xcodeproj` in Xcode 15+ and run on device or simulator.
