# Architecture Decision Records

ADRs capture **why** we made a significant technical choice, not only **what** we did.

## When to add one

- Choosing distribution (App Store vs direct).
- Sandboxing vs not.
- Major library or pattern (e.g. “SwiftUI-only vs AppKit panel”).
- Anything you’d explain twice to a teammate.

## Naming

`NNN-short-title.md` with sequential numbers: `001-sandbox.md`, `002-windowing.md`.

## How to write

Copy [000-template.md](000-template.md), fill sections, keep it to one or two screens.
