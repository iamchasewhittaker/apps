# Architecture Decision Records

ADRs capture **why** we made a significant technical choice, not only **what** we did.

## When to add one

- Choosing build tool or framework (e.g. Vite vs CRA).
- Sync strategy (blob vs normalized tables).
- Auth approach for mobile PWA.
- Anything you’d explain twice to a teammate.

## Naming

`NNN-short-title.md` with sequential numbers: `001-vite-vanilla.md`, `002-sync-strategy.md`.

## How to write

Copy [000-template.md](000-template.md), fill sections, keep it to one or two screens.
